# 🏥 XYZ District Hospital – Web-based Information Management System

## 📌 Project Description
XYZ District Hospital system is a web-based platform designed to help manage hospital services such as patient registration, appointments, pharmacy inventory, and health tips for the community.

The goal is to improve hospital service delivery and make information easily accessible.

---

## 🚀 How to Use the System
1. Open the website link (Vercel or local server)
2. Navigate through different pages using the menu
3. Fill forms like appointment or patient registration
4. Submit data to view results or store information

---

## 📄 Pages in the System (7 Pages)
1. Home Page – hospital introduction
2. Services Page – hospital services offered
3. Appointment Page – book doctor appointments
4. Patient Records Page – add/view patients
5. Pharmacy Page – manage medicine inventory
6. Health Tips Page – community health information
7. Contact Page – hospital contact details

---

## ⚙️ Features
- Appointment booking system (with Mutuelle number)
- Patient registration form
- Pharmacy inventory management
- Health tips posting system
- Navigation system
- Responsive design (mobile & desktop)

---

## ⚠️ Known Issues / Bugs
- Form data is not stored in database (if not implemented yet)
- Some pages may not be fully responsive on all devices
- No authentication system yet
